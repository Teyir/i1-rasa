## Commandes:


Start les actions:
```shell
rasa run actions
```
Dans un autre terminal:

Entrainez votre model
```shell
rasa train
```

Si vous souhaitez visualiser votre arbre:
```shell
rasa visualize
```

Lancer rasa:
```shell
rasa shell
```
